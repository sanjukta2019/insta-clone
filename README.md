#  Instagram Clone

A full-stack Instagram Clone built with **React**, **Firebase**, **Cloudinary**, **Chakra UI**, and **Zustand**. Users can create accounts, upload posts, like and comment on posts, edit their profiles, and authenticate with Google.

---

##  Features

-  User Authentication
  - Email & Password Login
  - Google Sign-In
-  User Profiles
  - Edit Profile
  - Change Profile Picture
-  Create Posts
  - Upload Images to Cloudinary
  - Add Captions
-  Like Posts
-  Comment on Posts
-  Delete Posts
-  Suggested Users
-  Responsive Design
-  Fast UI using React + Vite
-  Firebase Firestore Database
-  Zustand State Management

##  Installation

### Clone the repository

```bash
git clone https://github.com/sanjukta2019/insta-clone.git
```

### Go into the project

```bash
cd insta-clone
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

---

##  Environment Variables

Create a `.env` file in the project root and add your Firebase configuration.

Example:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=xxxxxxxx
VITE_FIREBASE_APP_ID=xxxxxxxx
```

Configure your Cloudinary credentials as required by your image upload utility or backend endpoint.
